# システム仕様
こちらのプロジェクトはWEBデータベースシステムです。
ユーザー情報の登録ができますので、組織のメンバー管理、学校や塾の学生情報管理、お店ではご客情報管理などができます。
情報登録は名前、パスワード、メールアドレスを使用します。
パスワードは暗号化されます。
